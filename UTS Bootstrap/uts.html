Tentang

Selamat datang di website ini. Terima kasih telah mengunjungi halaman ini, untuk mengenal lebih jauh tentang saya,izinkan saya untuk memperkenalkan diri. Saya Andjescha Skolastica seorang mahasiswi di jurusan Informatika. Lebih tepatnya saat ini saya sedang menempuh pendidikan di Universitas Sanata Dharma Yogyakarta, Fakultas Sains dan Teknologi, Prodi Informatika. Saya berasal dari Flores-NTT dan merupakan anak pertama dari tiga bersaudara.

Website ini dibuat dengan tujuan untuk mengaplikasikan pengetahuan dan keterampilan yang saya pelajari dalam semester ini dalam pengembangan web dan teknologi informasi secara umum. Proyek ini melibatkan pemahaman tentang konsep-konsep dasar dalam HTML, CSS, JavaScript,dan Bootstrap. Terima kasih telah meluangkan waktu untuk membaca tentang saya. Jika Anda memiliki pertanyaan lebih lanjut, masukan atau ingin berhubungan, jangan ragu untuk menghubungi saya melalui kontak yang tersedia di halaman ini.
